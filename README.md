# Multimodal and Agentic RAG for Grade 10 Mathematics Tutoring

This repository contains the initial implementation of a project that explores the use of Retrieval-Augmented Generation (RAG) for curriculum-aligned mathematical problem solving.

The project aims to develop a tutoring system for Grade 10 mathematics (Nepalese curriculum) that can generate structured, step-by-step solutions. It integrates multimodal capabilities to handle both text and diagram-based inputs, along with agentic workflows that enable the system to evaluate and improve its own responses.

The system combines:
- Retrieval-Augmented Generation (RAG) for grounding responses in textbook content  
- Multimodal processing for handling diagrams and visual information  
- Agentic workflows (e.g., Self-RAG, CRAG, Adaptive RAG) for improving reliability and correctness  

This is the **initial version of the repository**.  
Further details, implementation components, experiments, and results will be added as the project progresses.
